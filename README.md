# Topics Every Computational Scientist Should Know

This repository contains materials for a series of lectures
at Princeton where we'll cover some of the essential topics
every computational Scientist should know.

* Parallel computing ... with an emphasis on multithreading
on CPUs.  

* Floating point arithmetic and how to use it safely.

* How to use random number generators without making a fool
of yourself.

Code examples will mostly be in C.   All the C we use (and its
just a tiny subset of C) is covered in the file LearningC.c
